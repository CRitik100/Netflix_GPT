# Netflix_GPT

This is Repository for Netflix.

# Steps for building the project from scratch.

1. Intialize the package manger -> npm init
2. Install the parcel(Bundler) -> npm i -D parcel
3. Installing the React. -> npm i react react-dom
4. Add the script in package.json so that it will be easy to run the app.
5. Installing the Tailwind CSS-> npm install tailwindcss @tailwindcss/postcss
   - Then add below content in .postcssrc file.
     {
     "plugins": {
     "@tailwindcss/postcss": {}
     }
     }
    - Import the "@import "tailwindcss" in .css file

Note: 
1. node_modules contain all the installed packages.
2. dist is generated by parcel which is bundled, minified, and optimized code.
3. .parcel_cache is stores intermediate build files to speed up subsequent builds.
